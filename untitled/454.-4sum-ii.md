# 454. 4Sum II

{% embed url="https://leetcode.com/problems/4sum-ii/" %}

## Solution

```java
class Solution {
    public int fourSumCount(int[] A, int[] B, int[] C, int[] D) {
        Map<Integer, Integer> saveCD = new HashMap<>(); // save all the sum of c and d, <sum, occure times>
        for (int i = 0; i < C.length; i++) {
            for (int j=0; j < D.length; j++) {
                if (saveCD.get(C[i]+D[j]) == null) {
                    saveCD.put(C[i]+D[j], 1);
                } else {
                    saveCD.put(C[i]+D[j], saveCD.get(C[i]+D[j]) + 1);
                }
            }
        }
        int count = 0;
        // sum a and b, count a + b + c + d == 0
        for (int i = 0; i < A.length; i++) {
            for (int j=0; j < B.length; j++) {
                if (saveCD.get(0-A[i]-B[j]) != null) {
                    count+=saveCD.get(0-A[i]-B[j]);
                }
            }
        }
        return count;
    }
}
```
